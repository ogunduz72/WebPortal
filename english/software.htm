1.    Engineering software developed

    a. Flow models

  •         One dimensional (longitudinal) river flow model based on dynamic wave principle
  •         Two dimensional (horizontal plane), vertically averaged unconfined aquifer flow model
  •         Two dimensional (horizontal plane), vertically averaged confined aquifer flow model
  •         One dimensional (vertical) unsaturated zone flow model
  •         Two dimensional, vertically averaged overland flow model
  •         One dimensional infiltration model
  •         Dynamically interacting channel flow/aquifer flow model
  •         Dynamically interacting overland/unsaturated zone/saturated zone flow model

    b. Contaminant transport models

  •         One dimensional (longitudinal) river contaminant transport model
  •         Two dimensional (horizontal plane), vertically averaged unconfined aquifer contaminant transport model
  •         Two dimensional (horizontal plane), vertically averaged confined aquifer contaminant transport model
  •         Dynamically interacting channel contaminant transport/aquifer contaminant transport model
